QUINTA SEMANA

En la semana 5, tras llevar a cabo varias reuniones productivas, logramos definir un modelo preliminar para nuestro proyecto. Con este modelo predefinido en mano, comenzamos el proceso de corte láser. Este paso es crucial, ya que el corte láser nos permite obtener piezas precisas y de alta calidad, que son fundamentales para la construcción del prototipo. El uso de esta tecnología garantiza que cada componente se ajuste perfectamente, facilitando el ensamblaje y mejorando la funcionalidad del modelo final. A medida que avanzamos, seguiremos ajustando y perfeccionando el diseño para asegurar que cumpla con todas nuestras expectativas y requisitos técnicos.
